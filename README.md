# Construction Accounting & Ledger Web Application

## Overview
A professional, premium-styled accounting and ledger application designed for the construction industry.

## Tech Stack
- **Frontend**: Next.js, Tailwind CSS (White Black Gold Premium Theme)
- **Backend**: Python (FastAPI/Django) - *To be implemented*
- **Database**: PostgreSQL
- **Auth**: JWT

## Modules
1. **Client Module**: Manage clients and their projects.
2. **Project Accounting**: Dedicated ledger per project (Credit vs Expense).
3. **Vendor Module**: Manage vendors and project-based vendor expenses.
4. **Office Expenses**: Separate module for company overheads.
5. **Reports**: PDF generation with letterhead.

## Project Structure
- `/frontend`: Next.js Web Application
- `/backend`: Python Backend API (Planned)

## Setup
### Frontend
```bash
cd frontend
npm install
npm run dev
```
